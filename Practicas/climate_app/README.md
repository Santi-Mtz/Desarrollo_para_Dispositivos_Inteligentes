# Climate App

Es una aplicación hecha en Flutter que sirve de ejemplo para mostrar el clima de forma manual con una interfaz simple para teléfono.

## Qué es esta app

Es una práctica de Flutter que presenta una pantalla principal con temperatura, ubicación e icono del clima.

## Cómo ejecutarla

1. Instala las dependencias con `flutter pub get`.
2. Ejecuta la app con `flutter run`.
3. Para apresiar el cambio con el hot reload se debe ejecutar como debug console

## Resultado esperado

La app debe abrir una pantalla con el texto de clima actual, la temperatura, la ubicación (Santiago de Querétaro) y un icono representativo en este caso un sol.
